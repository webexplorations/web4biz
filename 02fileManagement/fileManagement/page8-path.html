If one or more photos are not showing up on the web page, check the path to see what folder the file should be in.

In the example on the left you can see that the file blanket.png should be inside a subfolder named cat which is inside of the folder named pet which is inside a folder name graphic which is inside the root directory.

A simplier way to state this is to use the "path":

rootDirectory/graphic/pet/cat/blanket.png

Notice how the path consists of a series of folders and subfolders and ends with a specific filename.
