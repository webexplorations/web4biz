Open up a window to view your directories and files.

In Windows - Open up File Explorer.
You can either right-mouse click on the Windows icon and select "Open Windows Explorer"

or

If the Folder icon is showing, just click on it.
 

Windows Explorer has two panels.

On the left is a list of directories (also called folders) and drives.

On the right is a list of the directories and files located inside the item that is selected in the left-side panel.

You can change the view by clicking on the List icon located in the top right corner of the Explorer window.

You can use the find field (upper right corner - look for the magnifying glass icon) to locate a file on the drive you have selected.
