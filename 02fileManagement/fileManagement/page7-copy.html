There are several ways to move and copy files. Here is a video to show you how.

Copy - Makes a duplicate copy of the file. The original file stays where it was.

Move - Automatically deletes the original copy of the file and makes a new copy at the location you choose.

Shortcuts (Hot Keys)

CTRL A - Select everything in the active window.

CTRL C - Copy the selected items and save them in the clipboard (the computer's temporary memory location.)

CTRL V - Paste what ever is in the clipboard into the place where the cursor is showing.

CTRL Z - Undo. A great shortcut to know!
