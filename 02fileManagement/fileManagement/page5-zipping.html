Many times you may want to share a group of folders and files with another person. Instead of sending individual files you can zip or compress them together into a single file. Both Windows and Mac have zip utilities built into the software.

In Windows:

Select the folder/files you want to include in the zip file. (See the next page in this tutorial for some tricks on selecting files.)

  • Right-mouse click on any of the selected items
  • Select the option "Send To"
  • Select the option "Compressed (zip) Folder"

All of the files inside the root folder will be automatically included in the compressed/zipped file.

The zipped file will have the name of the last item you selected. You can rename this by right-mouse clicking on the new zipped file and selecting "rename".

The new file will have an folder icon with a zipper going down the middle. (If you have file extensions turned on you will see the new file has an extension of .zip.)

Be careful, you can look inside a zipped file by clicking into it and it will appear as a normal, uncompressed folder. However, all the files will actually be compressed and will not "work" as expected.

To unzip select the .zip file, right-mouse click and select on "Uncompress"

These videos show how to zip and unzip files using Windows 7.

On the Mac:

  • Select the folder/files you want to include in the zip file.
  • CTRL click on any one of the selected items

The zipped folder will normally have the same name as the original folder or file with a .zip extension. Or it may be called archive.zip if your Mac is unsure which name to use.

To unzip simply double click on the .zip file and a new folder with the same name will be created containing all the uncompressed files and folders.
