You can make new folders by selecting the main or parent directory. Use the menu item File/New Folder to create a new folder.

The name of the new folder will be selected so all you have to do is start typing in order to give it the name you want.

Make the following set of folders. Notice that they are all in the main folder named "rootDirectory". The "root" folder is the parent directory. It holds all the other folders and files.

Notice the naming conventions:

  • Start your filenames with lower case letters
  • Don't use spaces.
  • New words start with a capital letter for readability.

What's the difference between a file and a folder?

A folder (or directory) is a container.
Folders contain other folders and files.
Folders do NOT have file extensions.
Each item in the graphic above is a folder.

A file is the data or information the computer uses.
Files have extensions.

For example:

  • myResume.docx
  • picOfJohn.jpg
  • logo.png
  • groceryList.txt
