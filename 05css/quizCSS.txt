48. Why won't this CSS code work?
body {
color:      #008000
background: #FEFEFE
margin:     20px 10px
padding:    2em
}
a. body is not a valid selector.
@@ body is a very valid selector. It is used to set defaults styles for the entire page such as font-family, font-size, and font-color. 
b. background: always requires multiple values such as url( ) and no-repeat.
@@ background: can have multiple values but it isn't required.
c. The selector should be using parenthesis ( ) instead of curly braces { }
@@ The curly braces are correct. 
*d. Each of the properties are missing a semi-colon ";" at the end.
@@ Those darn missing semi-colons will break all the lines that follow. Don't forget them!
e. em is not a valid measurement for padding
@@em is a very valid measurement and allows the padding to dynamically change as the size of the browser window changes.
f. margin and padding must have at least four values for top, right, bottom, and left.
@@margin and padding can have four values but it is not required. Two values designate top/bottom and left/right. One value uses that same measurement on all four sides of the object.
