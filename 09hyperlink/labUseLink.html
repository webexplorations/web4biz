WebDev - Learning Lab

Lab: Using Hyperlinks

Introduction

In this lab you will be building a mini-test site to experiment with linking and downloading files.

This Learning Lab will help you discover the following:

  • Create hyperlinks to other files on your site.
  • Create anchors on a page.
  • Create hyperlinks to anchors located on the same web page.
  • Create hyperlinks to other pages.
  • Create hyperlinks to anchors on other pages.
  • Create hyperlinks to pages out on the Web.
  • Create a link that will download files for the user.

Create a Folder and Three Files

  1. 09hyperlink folder - (You may already have created this folder during the first week of this course.)
  2. index.html - Inside this folder create a basic web page named index.html or copy and paste an existing web page from your site into this folder.
  3. longPage.html - Create a new page from your HTML template containing several headings and LOTS of text. You can use Lorem Ipsum to fill the page up with dummy text.

    Lorem Ipsum Tip: Right-mouse click on the generated page and select "View Source". Scroll down to the actual Lorem Ipsum paragraphs and copy those. That way you won't have to add your own <p> elements to the text.

    Half way down this page create an <h2> heading with the text, "My Favorite Things".
  4. A non-HTML File - Find a file that the user might want to download from your page. This could be a Word file, a PowerPoint, or and Excel spreadsheet. Any type of file that does not end with .html, .gif, .png, or .pdf. will work. Put this file in the same folder as your index.html file.
    • .exe - An executable file (a program that runs such as an installation program.)
    • .xls - An Excel file
    • .ppt - A PowerPoint File
    • .doc or .docx  - A Word Document
    • .zip - a compressed file

Add Text to index.html

Add the following HTML items to 09hyperlink/index.html

  • An <h1> describing your page - For example: Peter Johnson's Home Page
  • An <img /> element displaying an image
  • Text to fill the page. We need to give the page some depth so you can see the anchors at work. You can use any content you want, including Lorem Ipsum.
  • An ordered list containing each of the following lines of text:
    1. View a very,very long page.
    2. View a very,very long page opening a new tab.
    3. Go to the bottom of this page.
    4. View "My Favorite Things"
    5. Jump to South Central College website.
    6. Download this file (Give the name of the file).

Link to another page on the same site

On index.html, edit the list item "View a very long page", adding in a hyperlink. The light-gray code should already be on the page.

   <a href="longPage.html">
   View a very,very long page.
   </a>

This code will replace the text inside of the <li> </li> tags.

Save the page and view it in a browser. When you click on the link it should display the long page in your browser.

Link to another page on the same site opening a new tab.

Edit the second list item "View a very long page opening in a new page", replacing the text with this code. (The light-gray code should already be on the page.)

   <a href="longPage.html" target="_blank">;
   View a very,very long page opening a new tab.
   </a>

This is the same hyperlink, adding the attribute target=" ".

Save the page and view it in a browser. When you click on this link it should display the long page in a new tab.

2 Ways to Create Anchors.

Using the <a> element

In index.html, right above the <\body> tag, add an anchor: 

   <a name="bottomOfPage"></a>

This is a hidden anchor because it doesn't have any text to display.

Using the id=" " attribute

In longPage.html add an id="favorite" attribute to the <h2>My Favorite Things</> heading. 

   <h2 id="favorite">My Favorite Things</p>

Link to an anchor.

In index.html, edit the third item in the list, replacing it with a link to the anchor at the bottom of the page. (The light-gray code should already be on the page.) 

   <li>
   <a href="#bottomOfPage">
   Go to the bottom of this page.
   </a>
   </li>

Save the file and view the page. When you click on this link the browser should jump to the bottom of the page. If you don't have enough text on the page to show the change use CTRL + to make the text larger.

The browser treats the anchor as an id, or a unique place marker on the page.

Link to an anchor on another page.

In index.html, edit the fourth item in the list, replacing it with a link to the anchor you added to the long page. 

   <li>
   <a href="longPage.html#favorite">
   View "My Favorite Things"
   </a>
   </li>

Save the file and view the page. When you click on this link the browser should jump to the longPage.html and then down to the h2 heading, My Favorite Things.

Link to a page on the Web.

In index.html, edit the fifth item in the list, replacing it with a link to South Central College's website. 

   <li>
   <a href="http://southCentral.edu">
   Jump to South Central College website.
   </a>
   </li>

When the browser sees the text http:// it knows to go out on the Web to find that domain/filename. In this case it would be a file named index.html on the domain SouthCentral.edu

Links to Download Files

In index.html, edit the sixth item in the list, replacing it with a link to the non-HTML file you have in the folder. For the href=" " attribute use the actual filename of your file including the extension. 

   <li>
   <a href="replaceThisWithTheNameOfYourNon-HTMLFile.???">
   Download this file (Give the name of the file).
   </a>
   </li>

When the browser sees a file that it does not know how to display, it opens a file/save input window for the user, or automatically downloads the file into the user's download folder, depending on how her or his browser is set.

Save the file and click on the link. Look for the file in your download folder using Windows Explorer.

Links Around Images

In index.html, locate the <img /> element. Treat it like text, wrapping an <a></a> element around it.

You will be using the same filename for both the href=" " attribute in the <a> tag and the src=" " attribute in the <img /> element.

Under the image add a paragraph telling the user to click on the image. (The light-gray code should already be on the page.) 

   <a href="nameOfYourImage.png" target="_blank">
   <img src="nameOfYourImage.png" 
        style="width:100px;border:purple solid 1px;" 
        alt="description of your image" />
   </a>
   
   <p>(Click on the image for a larger view.)</p>

When the user clicks on the image it triggers the hyperlink, displaying the actual image file in a new tab. In-line styling was used for the <img /> element so the difference is obvious. Adjust as necessary.
