Grouping Elements with the <fieldset> & <legend>

Use the <legend> to chunk your form into groups

If you are presented with a list of 50 seemingly unrelated checkboxes on a survey it can be very daunting and will frighten most users away.

You can use the <fieldset> element to chunk the form into groups.

For example, you could break the 50 questions into 5 clearly groups that relate to each other. This will greatly increase the usability of the page for all users.

Use the <legend> element to identify each fieldset.

Here is an example of the <fieldset> and <legend> elements.

Here is the code:

<fieldset>
<legend>My Radio Buttons</legend>
<input type="radio" id="rad0" name="rad" />
<label for="rad0">Option 0</label>
<input type="radio" id="rad1" name="rad" />
<label for="rad1">Option 1</label>
<input type="radio" id="rad2" name="rad" />
<label for="rad2">Option 2</label>
. . .
</fieldset>

CSS can be used to change the outline or character formatting. You can either use the in-line style or use an id or class designation in the <head> section of the page.

<fieldset style="width: 300px;

display:inline;

border:2px solid #000099;

padding:6px;

vertical-align:top">

<legend>My Radio Buttons</legend>

<input type="radio" id="rad0" name="rad" />

<label for="rad0">Option 0</label>

&nbsp;&nbsp;&nbsp;

<input type="radio" id="rad1" name="rad" />

<label for="rad1">Option 1</label>

<input type="radio" id="rad2" name="rad" />

<label for="rad2">Option 2</label>

</fieldset>

Notice the how the ~&nbsp; non-breaking spaces were used to separate the items making it clear which circle goes with which label onscreen.
