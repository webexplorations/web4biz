What does <label> do?

Helping Screen Readers Determine Which text goes with which label

The label is a signpost. It tells the browser, "This text box is called firstName.''

Here's what the code looks like:

<label for="firstName">First Name</label><br />
<input type="text" name="firstName" id="firstName" size="20"/>

This creates an unambiguous (non-confusing) link between the text that displays on screen "First Name" and the input textbox which has the attribute id="firstName".

Other Benefits

An added benefit to using a label is a large area is established on screen. This is especially useful when using radio buttons and check boxes. The user can click anywhere on the label to select the item.

id=" " means there will be only one per page

The key to all of this is the id=" " attribute. Each id=" " attribute must be unique for that page. This allows the label to know exactly which item it is attached to.
