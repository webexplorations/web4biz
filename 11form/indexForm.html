Forms
Getting input from the user.
See Checklist View

In this module you will learn:

Use forms to get information from the user.

  • Create a form on a web page using the <form> tag.
  • Consistently use standardized prefixes for all form object names.
  • Demonstrate using input boxes within a form.
  • Demonstrate using radio buttons/option buttons within a form.
  • Demonstrate how to use selection lists within a form to obtain consistent input from users.
  • Demonstrate using check boxes for yes/no input within a form.
  • Demonstrate using text areas within a form for free-form input from users.
  • Demonstrate using submit button to handle the information collected by a form.
  • Demonstrate at least three form elements introduced with HTML5

WII-FM (What's In It - For Me)

Forms are your client's connection to the user. Forms are used for e-commerce, feedback pages, and almost all communications from the user. This module will show you how to create all the basic elements of a form as well as how to automatically send an email with the form information.

Forms are important for more advanced topics in web work and you will be using these elements a lot in the future as you build more complex web pages.
