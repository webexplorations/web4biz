Web Forms - An Overview

The <form> Element

The <form> element encloses all of the objects such as textboxes, buttons, and drop-down boxes. A form can include dozens of objects or just one. CSS and even tables can be used to help organize the form objects on a page.

The <form> tag is used within the <body> element of a web page.
