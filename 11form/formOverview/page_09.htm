Radio Buttons (opt)

Radio Buttons allow a single choice to be made from a list of choices. (If multiple options are wanted, then use checkboxes.)

And here is what the results will look like:

The name attribute is critical for radio buttons. Notice that it is the same for each radio button object. The browser uses the name to determine which radio buttons should be grouped together. With that information the browser automatically turns off radio buttons as others are selected by the user.

checked="checked" Add the attribute checked="checked" as part of the <input> element if you want a specific radio checked when the form is first displayed. If you don't include a checked attribute in one of the elements, none of them will be selected.

There's one other tricky piece about radio buttons. The value of the value attribute is what is sent to the server. Don't be fooled thinking the text after the <input> tag is sent, that is only for the end-user.
