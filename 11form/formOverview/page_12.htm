Text Area (txa)

The textarea box allows the user to input multiple lines of text. It is perfect for the typical comment box.

rows=" "
cols=" " The attributes for rows and cols allow you to determine how many lines of visible text will display on the page. If the user wants to type in more lines, the textarea will automatically scroll.

If you want text to display inside the textarea, add it after the start tag <textarea>. Keep it short though, because the user will have to delete your text before typing their own.

This is a two-tag element, so don't forget the ending tag </textarea>!

Add a textarea without any text showing to your demonstration page. Try it first with 100 cols to see how the width affects the table columns. Then change it back to 50 cols:

Normally the <textarea> and </textarea> tags are put together without any space or lines between them.

If you put them on separate lines the browser will add spaces inside the text area and the cursor will sit in the middle.
