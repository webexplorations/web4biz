Web Forms - An Overview

Web Forms

Forms allow you to start a conversation. Without forms you are only able to talk to the user, and they can only listen or read.

By adding forms to your web pages you have the ability to:

  • Collect data from your customers
  • Allow the user to make decisions and customize what feed-back they receive
  • Allow users to change the web site environment
  • A customer order form is an example of data collection.

Making several selections from drop-down boxes to determine which data the customer wants to view shows how forms can determine what information the customers should see. CarSoup.com is an excellent example of this use of forms.

Adding a single-button form to a page will allow users to change their environment, from frames to no frames, a different background color, turning music on and off, watching a video, etc.
