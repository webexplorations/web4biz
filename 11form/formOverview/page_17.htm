Web Forms - An Overview

Set the Tab Order

American and European users expect to TAB through a form from left to right, top to bottom. Often if you add a new object in the middle of your form it will be in the wrong tab order.

tabindex=" " Add the tabindex=" " attribute to each of the controls on the form. This will give you the control you need to decide which field becomes active when the user hits the TAB key.

For example: <input name="fname" tabindex="1" />
