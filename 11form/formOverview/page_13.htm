Hidden Fields (hdn)

Hidden fields are often used to send data to the server that the user doesn't know or care about. For example, a contact form on different websites sends information to the server which than sends out emails to specific people with the user's questions. How can the emails be automatically set up to show which site the email question originated from?

By using a hidden field this information can be sent to the server and included in the subject line and content of all the emails sent out.
