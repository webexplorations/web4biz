Drop Down Lists (lst)

Selection or drop down lists can allow the user to select an item from a list.

Unlike the elements we've used so far, this is a two-tag element that surrounds each of the <option> that should be included in the list.

size=" " The number of items displayed is determined by the size attribute. A value of "1" will make the list appear as a drop-down list and can be very useful when you don't want to waste real estate on the screen.

multiple="multiple" If you want the user to be able to select several options, add the attribute multiple to the <select> element. The user can select the items by holding the control key down and clicking each item to toggle it on or off.

selected="selected" can be used in the <option> that should be selected by default with the form is first displayed.
