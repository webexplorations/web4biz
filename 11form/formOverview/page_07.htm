The Password Box (pwd)

The password text box is used to prevent others from seeing a password as it is typed in. It acts just like a textbox except that the characters display as shapes so they are unrecognizable.

Keep in mind that this is a display mechanism only. When the password is sent to the server across the web it is sent as normal text and can easily be extracted using a packet sniffer. If you are sending critical information such as passwords from your web page forms, make certain that the data is encrypted. This will be done by a server-side language and special Web connections such as a Virtual Private Network (VPN).
