The text box allows the user to type in single line entries. This is perfect for name, address, and phone numbers.

Here are the attributes you can use to customize this object:

type="text" tell the browser that this is a text box.

name=" " allows you to name the object so it can be referred to by the CGI script. (You should use the name attribute for every object inside a form.) Keep in mind that spaces are not allowed. Be smart and keep your names all lower case except for word boundaries and don't use plural names.


size=" " tells the browser the length of the text box in characters.



maxlength=" " tells the browser the maximum number of characters you will allow. If the maxlength is longer than the size, then the textbox will scroll as the user types in characters.


value=" " will assign a string value to the text box. This helps non-visual browsers but is an irritation for visual users because they have to delete the text before they can type their own text in. If you use value then keep it short.

Notice how the attributes can be stacked for readability. Remember, web browsers ignore the line returns in the code unless they find a <br /> or a <p> </p> tag.

Also, the value=" " attribute is used to supply a default value in the textbox. This is a real time-saver when programming with forms. All the programmer has to do is hit Enter for each form object and he or she doesn't waste time typing in information every time they run the page. Later, when the web page goes "live" this default value will be removed.
