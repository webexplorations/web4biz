Web Development - Lists

Summary

At this point you should be able to do the following on any web page you write:

  • Create a bulleted (unordered) list
  • Create an ordered list that is numbered, lettered, or uses Roman Numerals
  • Create a list inside of a list using valid XHTML coding.
  • Use CSS to use custom images for bullets
  • Have a debugging checklist in your head that will help you correct some of the most common problems with lists and CSS.
  • Have a basic understanding of the three parts and punctuation of a CSS selector.
