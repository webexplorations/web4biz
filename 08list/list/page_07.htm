CSS Punctuation

Just like a sentence in English follows specific punctuation rules, each selector uses a set of curly braces { } to "frame" the list of properties associated with it.

Each property ends with a colon ":" separating it from the value.

Each value ends with a semicolon ";" designating the end of the line.

The url( ) tells the browser to look in the file system for the file that is designated.

Commenting is different inside the <style> element. You have to start the comment using /* and end it using */

Be generous with your comments, especially on special items (such as color names). The person you help in the future may be yourself!

Debugging Tip

If the custom bullets do not appear in your browser go through this checklist:

  1. Did you spell everything correctly? (particularly the name of the bullet graphic).
  2. Do you have the colons after the property and the semicolons at the end of the line?
  3. Are the curly braces in the correct places: one after the selector and one at the end of the entire rule, boxing in the property and its value.
  4. Is the bullet graphic file in the same folder as the web page?
