Using the type=" " attribute

You can easily change the display of the list by using the type=" " attribute. This can be used for either element: <ul>, <ol>, or <li>.

For the <ul> you can use type="disc", type="square", or type="circle".

For the <ol> you can use type="1", type="a", type="A", type="i", or type="I"

For the <li> you can use any of these.

Note that the value of every attribute must be enclosed inside of quotes.
