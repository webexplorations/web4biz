Debug Tip

Use the HTML Validator FireFox addon. This will show you any validation errors or warnings that you may have with your code. If you don't have any errors than you know your code will display in all standard browsers, from cell phones to desktop browsers.

Here is an example of the HTML Validator at work.

If you don't put the sublist in the correct place you will get validation errors that display in the lower right corner of the screen.

If you double click on the information that look like this:

Notice how the sublist in the error code is outside of <li> pair. It will confuse most browsers causing the lists to possibly display incorrectly or not at all.

It is much more efficient to fix errors as they happen than later, after the web page is all finished.
