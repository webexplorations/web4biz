Debug Tip

Changes not showing up in the browser?

  1. Make sure you are saving the file each time you make a change (CTRL S).
  2. Make sure you are viewing the same file you are editing.
    Add some weird but noticeable characters to the code such as XYZ.
    Save the code and view the file in the browser.
    If the XYZ does not show up than you are probably looking at a different file.
