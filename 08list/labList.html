Practice Lab: List

This lab will give you practice with the following HTML skills:

  • Display a bulleted list on a web page.
  • Display a lettered list on a web page.
  • Utilize at least three special characters on a web page.
  • Add a horizontal rules on a web page for increased readability.
  • Use the <img /> element to display graphics on a web page.

What to do:

The board of directors for Money Plus has hired you to create a web page for them. They want the page to list the requirements for their financial aid package. They have agreed to pay you $100.

Important: As you complete each step, save the page and view your changes in the browser checking for errors.

(1) Create a basic web page starting with a comment header block and using the essential html elements:

<!DOCTYPE html>
<html>
<!--
financial.html - Lab: List to explore my knowledge of HTML lists
Written by: Add your name and email address here.
Date:
-->
   <head>
      <title>     </title>
      <meta charset="utf-8" />
   </head>
   <body>
       All the display content for the page goes in the <body> element.
   </body>
</html>

(2) Name the file: financial.html -- Don't forget to set the file type to "HTML" or your text editor might save the file as a ".txt" file.

Debugging tip: If the color coding does not show up and all the text in your text editor is black, than you probably have the file extension wrong. Look at the filename tab at the top of the editor to see the actual name of the file you are editing.

(3) Add the title: "Money Plus Financial Aid Program" so these words display in the title bar of the browser and on the Start bar when customers are viewing the web page.
Put this information in the <title> </title> section of the web page.

(6) Save this image to the same folder as your html document by right-mouse clicking and choosing "save image as". Name this image money.jpg. Remember to always follow the naming conventions. (Start with lower case letters, useCamelCase, no spaces or underscores, all file extensions are lower case.)

Or.... be super professional and save the image in a graphic sub-folder!

(7) Display the image on the web page under the level 3 subheading. For the alt="" attribute use: "photo of $100 dollar bill". Need help? Click here.
Note: If you saved the money.jpg image in a subfolder you have to tell the browser the name of that subfolder. For example, if the subfolder is named graphic: <img src="graphic/money.jpg" alt="photo of $100 dollar bill" />

(8) Display a horizontal rule across the page. Need help? Click here.

(9) Add a level 2 heading that says "Financial Assistance Available"

(10) Display the following paragraph in bold text: "To be eligible for this program, you must meet the following criteria:"

(11) After the paragraph, display the following list using round bullets: Need help? Click here. Keep in mind that the <p> element and the <ul> and <ol> elements are all block elements. You can't put a block inside of a block. They have to be kept separate!

Have three children or more children under the age of twelve.
Have an income below the federal poverty line.
Be gainfully employed.
Not have any legal convictions.

(12) In a new paragraph, display the following in bold: "To apply, please do the following:"

(13) After the paragraph display a numbered list with the following three items:

(14) Add a sublist after the second item in the numbered list (Step #13 - "Assemble the following papers.") start a new lettered list with lower case letters displaying the following items:

Tax records or a certificate of your income level.
A letter from your employer.

Make certain that this sublist is INSIDE of the <li> and </li> element. If you put it after the second </li> the HTML validation program will give you lots of errors.

Need help? Look back in the Tutorial: Lists near the end to see how to create a sublist.
