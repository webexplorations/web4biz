Lists and Paragraphs are Blocks

A block element can't be put inside another block element.

This is a paragraph introducing the following bulleted list:

  • item one
  • item two
  • item three

Paragraphs are block elements.

Lists (both <ol> and <ul>) are block elements.

You can't put a block inside a block, so always close off a paragraph using </p> before starting a list.
