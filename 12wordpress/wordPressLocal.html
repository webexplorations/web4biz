Install WordPress on your localhost

Use WAMP (Windows-Apache-mySQL-PHP) or MAMP (Macintosh-Apache-mySQL-PHP) to set up a local web server stack on your computer.

WAMP (Windows)

  • Put it in the root of C: drive.
  • Put all your web files in the wamp/www/ folder. (just like public_html on the t.server)
  • Set up a database named: wordpress
  • DBF User name: root
  • DBF Password: nothing/empty/no text

MAMP (Macintosh)

  • Put it in your Application folder
  • Put all your web files in the mamp/www/ folder. (just like public_html on the t.server)
  • Set up a database named: wordpress
  • DBF User name: root
  • DBF Password:root

Installing WordPress

  1. Download latest version of Wordpress WordPress.org

    Save this file to your flash drive in case you do a re-install in the future.
  2. Unzip the file.
  3. Select and copy all of the inner files (will contain folders: wp-admin, wp-content, wp-includes)
  4. Copy these into the www folder for WAMP/MAMP.
  5. View the page in browser: WAMP (Windows: localhost/ MAMP (Mac): localhost:8888/
    This will start the install process.
  6. Use the userID and password for the database (listed above).
  7. To access the administration forms use localhost/wp-admin localhost:8888/wp-admin
  8. In another browser open up the user side of Wordpress: localhost or localhost:8888
