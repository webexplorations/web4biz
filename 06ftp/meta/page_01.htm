Meta Tags

Meta means about.

Meta thinking is thinking about thinking.

Meta data is data about data.

Meta tags contain information about the tags or information about the web page.

When search engines catalog a web page they look for the <meta name="description" /> meta tag.

The text in this tag is used as descriptive text when the link to the page is displayed in search results. No description meta tag? Google, and other search engines, try to decide what text from your page to display...

Meta tags are also useful to list the copyright owner of a page, tell the search engines to ignore the page, as well as redirect the user to another page.

In this tutorial you will learn how to write meta tags that will determine how a page is handled by search engines
