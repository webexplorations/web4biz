Web Development - Meta Tags

Summary

This short tutorial has given you a quick overview of meta tags. You should be able to answer the following questions:

  1. What is the meaning of the word meta? Can you give an example where this prefix is used with other words?
  2. How can you control the paragraph that prints in the search engine listings?
  3. How can you keep search engines such as Google from indexing or searching a page?
  4. Are meta tag keywords important in today's Web world? Why or why not?
  5. List at least two other things that meta tags can describe about a web page.
