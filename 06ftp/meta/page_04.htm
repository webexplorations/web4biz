Web Development - Meta Tags

<meta name="refresh" />

Use the name="refresh" meta tag when you want to jump the user to another page.

For example, imagine having to re-arrange your site, or convert it from regular XHTML to PHP files. If the page has been established, a lot of people out on the Web may already be linking to it. These links are invaluable for the SEO of the site so you don't want them broken.

The solution is to re-edit the original page so it has a "Redirecting you to the new location" message and set a refresh meta tag.

Here is some example code:

 
<meta http-equiv="refresh" content="5; url=newurl.php">

The number is the number of seconds in the content=" " attribute is how long the user will have to wait before the page is redirected. Setting it to 0 results in an instant redirect.

Note how the content=" " attribute includes two bits of information, the amount of time and the url of the new page. Both of these have to be included inside the same set of quotation marks " ".
