Web Development - Meta Tags

<meta name="keywords" />

In the early days of the Web, before Google, search engines used the keyword meta tag to determine which categories the page should be listed under.

Web programmers quickly found ways to manipulate the system by including competitors names, and repetitions of the same keywords. This tag is no longer considered useful and most search engines ignore this tag, using more sophisticated algorithms to determine the keywords of a page.

Here is an example from the Computer Careers site showing the keyword meta tag in use:

 
<meta name='keywords' content='computer careers, college, two year degree, web programming, system i, system z, as/400, i-series, RGB, networking, web development, linux, certificates, degrees, XHTML, CSS, SEO, XML, Java, VB.NET, Excel, Access, computer programming, security, mainframe, COBOL, HTML' />
