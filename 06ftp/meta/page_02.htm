Web Development - Meta Tags

Four things to know about meta tags.

(1) Meta tags are special elements that are written in the <head> section of each web page. They contain information describing the page. With the prefix "meta" you can think of them as "tags about tags". None of these tags change how the page actually looks.

(2) Notice that each meta tag follows the same pattern. They start with <meta name=" " /> and then use different attributes based on what they are describing.

(3) These are single element tags, so to be XHTML valid you must include a space, slash before the closing bracket: <meta name=" " content=" " />

(4) Use meta tags on every single page. Every page is handled as its own entity by the search engines. Be smart and include meta tags in a template and then customize them based on the page content when necessary. This will give all your pages consistent information and save you a lot of typing.
