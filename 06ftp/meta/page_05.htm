Web Development - Meta Tags

<meta name="robots" />

The robot meta tag allows you to control if the page will be catalogued by the search engines.

The best solution is to use a file named robots.txt in the root directory of the site. This meta tag allows another layer of control in the event you are not able to upload a separate file.

The list of possible instructions include:

noindex - the page will not be indexed by the search engine. Note that there are no spaces between these two words.

index - the page WILL be indexed. (This is the default instruction for any page so it is optional.)

nofollow - the search engine robot will not follow any links contained on the page

follow - indexes on the page will be followed. (This is the default instruction for any page so it is optional.)

Here is an example that will keep the page off the search engine listings:

 
<meta name="robots" content="noindex, nofollow" />

If this meta tag is missing then the robot terms will be assumed to be "index, follow" (e.g. "all"). If the keyword all is found in the robots terms list it overrides all other values.

Where would this be useful?

One example is the Web Development course Student Showcase. (http://southcentral.edu/cc/course/showcase) If these web sites did not include the noindex, nofollow attributes the search engines would be including them.

Meanwhile, the clients may be having their actual live sites updated. If both sites are listed several customers will be seeing the old information contained in the student showcase.

This happened with a Used Car Lot that was in the showcase several years ago. The client kept getting calls about a Mustang that had been sold a year earlier. They customers were clicking on a link to the showcase page, seeing the Mustang highlighted on the front page, and calling the used car dealer for more information.

When developing a site for a client it is always smart to use the noindex, nofollow attributes until the site goes "live".

Just don't forget about this meta tag! Once the site is live and the client wants to establish a strong SEO you will have to set this meta tag to index, follow or it will never show up on the search engines!
