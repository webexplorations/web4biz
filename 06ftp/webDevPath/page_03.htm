Different Views

Windows Explorer will show you the contents of a folder several different ways. Each has its pros and cons.

You can change the view by clicking on the view icon in the top toolbar:

The Thumbnails view is great when you need a quick preview of different graphic files. It is very clumsy for general file handling work however.

The Tiles View is nice for a quick overview and is a little more compact than the thumbnails view.

The List View is nice and compact and will show all the available files and folders quickly and clearly.

The Details View has more information than the List view and is very good when you are checking date stamps and other details about the file.
