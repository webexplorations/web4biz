Using a <br /> - Line Break

Add in some <br /> elements in order to tell the browser to start a new line.

Remember the XHTML rules? Every start tag needs a stop tag.

The <br /> element is a single-tag element. In order to meet the XHTML requirements we add a space slash " /" to act as a "stop" tag.

For double tag elements the slash is at the front: <p> and </p>

For single tag elements the slash is tacked on at the end:<br />

Add some <br /> elements to the paragraph and view the results in the browser.

Notice how we used the special character ~&lt; to display the < character. If you us the < by itself the browser thinks it is a new markup command!
