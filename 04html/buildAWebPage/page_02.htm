Web Development - Build a Web Page

What Tools?

You need two tools to create web pages:

A color-coded Text Editor

Notepad++ (Windows) - You can run this from your flash drive.

TextWrangler (Macintosh)

A Browser

- FireFox is recommended the highest. FireFox is very compatible with current standards and has many great development tools.

- Google Chrome is also good and has many tools built in.

- Microsoft Internet Explorer (IE) is worst. It is not compatible with current standards.

Run both programs (the text editor and the browser) side-by-side so you can easily switch back and forth between the two.

Windows Tip: Right-mouse click on the Start bar and choose the horizontal display option.

Don't use a word processor like Word or WordPad. They add in hidden characters that will show up as weird text in the browser.
