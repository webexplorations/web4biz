How XHTML Works

The browser is always looking for the Less Than character "<".

When it sees one it knows that is the start of a markup command. After the < it reads each character until it gets to the Greater Than ">" and then it looks up the command in its dictionary to see what to do.

For example:

<h1> tells the browser to start displaying the text that follows as a main heading. It will continue to do this until it comes to the stop h1: </h1>

If you forget to include a </h1> the entire page will be displayed as a main heading.

After the browser has read a tag it discards it. Notice that none of the tags show up on the browser page.
