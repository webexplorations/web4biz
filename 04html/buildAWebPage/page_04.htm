Web Development - Build a Web Page

The Essential Elements

Here are the essential XHTML elements.

Notice that the markup tags come in pairs.

<html> and </html>

<body> and </body>

<p> and </p>

The first tag is called the start tag: <body>

The second tag is called the stop tag: </body>

Type this text into your editor.

Name the file: test.html

<html>

<head>

<title>This is a test page</title>

</head>

<body>

<h1>Test Page</h1>

<p>This is a test paragraph.</>

<.body.

</html>

To view the file in the browser use:

File/Open File/ locate the file test.html

Notice that you don't have to be connected to the Web. The browser is simply translating your text using the markup tags.

Here is a screen shot showing how the various "secret" codes appear in the browser:

The <title> element displays in the bar along the top of the browser as well as in the tab for that page. In Windows the <title> information also displays along the bottom in the Start menu bar.

Important: All of the information that appears on the browser page must be inside the <body> and </body> tags.

The <h1> is a header and displays as a heading in the page itself.

The text inside of the <p> element appears as a paragraph. The words will automatically wrap if the user changes the size of the browser window.
