Paragraphs and Line Breaks

Two things the browser ignores are line-returns and more than one space.

In your code break up the lines so there is only a few words on each line.

Save the file (CTRL s) and view it in your browser (CTRL r).

None of the changes show up in the browser!

All the browsers are programmed to ignore multiple spaces and new lines. This lets the programmer format code for readability without changing the look of the finished web page.

On the next page you will see how to use the <br /> to tell the browser to start a new line.

To add multiple spaces use the non-breaking space special character: ~&nbsp;
