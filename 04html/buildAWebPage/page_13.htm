Web Development - Build a Web Page

Summary

You should be able to answer these questions as well as write a basic web page and display it in a browser.

  1. What is the "magic" character that the browser uses to determine the beginning of a markup tag? What is the character that is used to determine the end of a markup tag?

  2. What are some of the single tag elements mentioned in this tutorial?

  3. What are attributes?

  4. What are the two software programs that you need to create web pages?

  5. Do you need to be connected to the Web to write or display web pages?
