Using <div>

The notes "Tony's Birthday" needs to be controlled. You can do that with a div tag.

In the code itself add a div tag around the text of any note. Make it a class so we can use this CSS rule for multiple div tags on the page. (An id # can only be used once on a page. A class . can be used multiple times.)

If you use consistent indenting it will be an easy matter to copy this div tag into another cell anytime in the future.

Here's the CSS rule that will format the notes in a nicer fashion.

Type this in the <style> element in the <head> section.

[D]

Note the dot right before the word "note". This tells the browser to use this rule on every class named "note".

Just remember: a dot "." signifies a class (which can repeat on the page) A hashmark "#" signifies an id. Each id must have a unique name on a page.

We reduced the font size, aligned the text to the right and added some padding on the left for longer notes.
