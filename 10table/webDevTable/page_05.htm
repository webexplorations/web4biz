summary=" "

HTML 5 Update:

The HTML 5 standard does not include the summary=" " attribute. It turns out the screen readers such as JAWS were ignoring the summary=" " attribute and standard visual browsers weren't showing the information at all (like they do the alt=" " attribute for images).

The summary=" " attribute is very important for the accessibility of every table. Non-visual browsers (and search engines) use this attribute to determine the purpose of the data in the table. It is very much like the alt=" " attribute for the <img /> element except that the text in the summary=" " tag is not visible to the normal user unless they are using a special browser such as JAWS or text-to-speech software.

Include this attribute as part of every table element you write.
