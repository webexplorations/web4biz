colspan - spanning columns

Often a table row needs to span across several columns. You can use the colspan=" " attribute to do this. The photo gallery is an excellent example.

Instead of having a separate column heading for each column you can span across all three columns with a single heading.

Replace the three <th> heading elements with this line:

<th colspan="3">Photo Gallery</th>

If you end up with some extra columns and have some empty spots in your table like this:

you'll have to go in and remove some of your code. The colspan=" " attribute creates as many columns as you say and if you don't delete existing columns (other <td> or <th> elements) you'll end up with extra columns.

The same technique can be used to span several rows using the rowspan=" " attribute.
