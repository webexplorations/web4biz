CSS and <tr>, <th>, and <td>

Let's extend your CSS skills to make the rest of the table look better.

Here's the CSS code you can add inside the <style> element on your page:

[D]

(1) A thin, dark-blue border, matching the border around the table itself is added.

(2) The height and width of each <td> cell is increased to be 5 1/2 times the established font size.

(3) Padding of 5px surrounds the text.

(4) The text is aligned to the top of each cell. Notice how the text automatically aligned horizontally to the left. This is the default if you don't specify a horizontal alignment.
