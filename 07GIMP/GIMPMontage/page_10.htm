Create a Montage Using GIMP

Add the Eagle to the Montage Image

To add the eagle follow the same technique you've done already:

  1. Select the eagle.jpg image using Window from the menu bar.
  2. Use the fuzzy selection tool and select the sky area. Set the threshold around 15. (Or, depending on your experience level, use a more sophisticated selection method to get more details like around the wing tips.)

  1. Invert the selection using Selection/Invert
  2. Copy the eagle using Ctrl C (or edit/copy)
  3. Select the montage.jpg image
  4. Create a new layer (layer/new) and name it "eagle"
  5. Select the "eagle" layer
  6. Paste in the eagle (CTRL v)
  7. Resize it (Shift T)
  8. Reposition it by clicking on the Move tool (four headed arrow) and dragging the layer.
