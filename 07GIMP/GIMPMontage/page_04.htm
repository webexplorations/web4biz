Create a Montage Using GIMP

Create a New File

Create a new file to hold the finished montage. Use File/New from the menu bar along the top of the tool panel. Set the size large enough, 800w x 600h. You can always reduce the finished version to a smaller size later.

If you open up the Advanced Options in the "Create New Image" dialog box you can also set the dpi and color type. 72dpi and RGB color are good choices for web images.
