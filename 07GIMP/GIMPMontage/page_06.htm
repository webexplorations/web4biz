Extract out the Image of the Girl

There are many ways to separate an image from the background or extract part of an image from the rest of a photo. One quick way is to use the Fuzzy Selection Tool.

Select the image of the girl, and with the fuzzy selection tool select the background.

Depending on the threshold value different parts of the background will be selected. You can either raise the threshold value or else hold the shift key down while you select other areas. You will see a small plus sign "+" as you move the cursor over the image.

[D]

We want the image of the girl not the background. Use Selection/Invert to change the selected area from the background to everything that you didn't select as the background, which is the image of the girl.

With the girl selected use Edit/Copy (Ctrl C) to save the image on the clipboard.
