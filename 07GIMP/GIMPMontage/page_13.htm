Create a Montage Using GIMP

Resources

Here are some valuable resources that will help you learn more about GIMP and make your photo editing sessions more productive:

Carol Spears - GIMP Quickies - http://www.gimp.org/tutorials/Lite_Quickies/

She shows you how to use GIMP to:

  • Change the size of an image (scale)
  • Make jpg files smaller
  • Crop an image
  • Find out stats about an image
  • Flip and/or rotate an image

The GIMP Handbook shows you how to accomplish whatever you want using GIMP. Nice clear instructions with lots of graphics.
http://docs.gimp.org/2.4/en/index.html
