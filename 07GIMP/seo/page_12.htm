Web Development - SEO

Write clean code

Valid web pages rank higher

By writing to XHTML standards you are making the search engine's job much easier. Using the rules of XML documents it knows that every start tag has a stop tag and every attribute is surrounded by quotes. It only has to look for lower case tag names.

By making certain every page passes TidyUI or the W3C Validation you are increasing the SEO for that page. That means zero errors and warnings.

Limit the Use of Tables

Tables are very tag heavy. Keep tables to a minimum by using CSS to position your blocks of code on the page. For the best SEO you want the fewest number of tags and the greatest amount of content.

You can further remove code bloat by moving all JavaScript to an external file. This is very similar to putting CSS in external files.

Use Text Links

Text links are always easier for a search engine to index. Graphic links, JavaScript links, and Flash all inhibit search engine spider programs. Instead, use CSS to make your text links "pretty" and interesting to the user.

Always provide a site map

A site map describes how the site is laid out. You can submit your sitemap to Google which will give the search engine more information about your site. In essence you will be assisting Google's spider programs by helping update the Google index.

There are many sitemap generators including sitemapgen out on SourceForge.
