How Search Engines Work

Spiders, Crawlers, and Robots

The most popular search engines such as Google and Yahoo use a program called a spider, web crawler, or robot. These names all mean the same thing. This type of program reads a web page and then follows all the links included on that page.

The spider extracts information from the site, using the HTML tags in the page to determine what is important. This information is stored in an index. When a person does a search the results that are displayed are pulled from this index based on mathematical formulas that determine what information is pertinent to the search string that was typed in.

Ranking Search Results

How the results of a search are ranked are the key to SEO. Google changed the search engine rankings by using a "voting" formula. A web page receives a higher Google rank based on how many other web pages linked to that page and how important those web pages were. Having a link to your page on ibm.com gives you a higher Google rank than one from computers4Sale.com.

Since Google's inception they have been continuously improving their search formulas (algorithms). Many of the concepts used by Google are now being adapted by other search engines as well.
