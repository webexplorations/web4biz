Use Robot.txt

Tell the Search Spiders or Robots Where to go

This file tells crawlers which directories can or cannot be crawled. Make sure it's current for your site so that you don't accidentally block the search engine programs.

One use of the Robot.txt file is to keep certain files from being indexed. This is a great trick while you are developing a site and have the pages on a temporary server. Don't forget to change the Robot.txt file when the site or page goes live!

Here's an example of this file:

# /robots.txt file for http://webcrawler.com/
# mail webmaster@webcrawler.com for constructive criticism
User-agent: webcrawler
Disallow:
User-agent: lycra
Disallow: /
User-agent: *
Disallow: /tmp
Disallow: /logs

The lines beginning with # are comments.

The * in the third User-agent is a wild-card which means every robot visiting.
