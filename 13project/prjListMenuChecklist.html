prjListMenu - 55 Total Points Possible

Create a web page - mySkillSet.html - 5 points

Include a thumbnail image (50-150px wide). If the user clicks on this image the browser will display a larger view of the same graphic.

Include an unordered list on the page with these items:

  • Using a numbered sublist
  • Using special characters
  • Downloading non-HTML files
  • Link to an anchor on the same page
  • Link to an anchor on another page on the site
  • Link to your t.server site

Create another web page - aboutMe.html - 5 points

An anchor for each subhead on the page using a hidden anchor or the id=" " technique.

Demonstrate the following on your pages - 20 points

Each of these items will have an anchor to it so you can jump to it from the list on mySkillSet.html.

  • 2 points - Using a numbered sublist correctly
  • 1 points - Using at least 3 special characters
  • 2 points - Downloading a non-HTML file
  • 3 points - Link to an anchor on the same page
  • 4 points - Link to an anchor on another page
  • 3 points - Link to your t.server
  • 5 points - Linking to a larger image from a thumbnail-size image

Create a Menu using an external CSS file using the technique used in the Learning Activities - 15 points

On each page of your web site include a menu using CSS linking to all the other pages on the site.

The CSS should be in an external style sheet named prjListMenu.css

All pages must be valid HTML code using the W3C Validator. - 5 points
