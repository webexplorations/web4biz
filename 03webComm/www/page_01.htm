The Web

This tutorial will help you understand what happens when you click on a link and the browser displays a web page.

You've seen what the Internet looks like, and you've seen how the client/server concept makes file sharing possible between lots of different web servers around the world (as long as they are connected to the Internet or some other network.)

In this tutorial you will be walking through the steps a web request takes, tracing the journey from browser to a web server and back again.

The browser on your machine is considered a client application. And the web server is just that: a server. This is client/server in action!
